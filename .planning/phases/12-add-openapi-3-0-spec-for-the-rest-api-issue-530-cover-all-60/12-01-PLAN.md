Create the drift-detection and schema-validation tests so the contract exists from the start.

Purpose: De-risk the phase. The spike is the single biggest scope unknown; the pipeline is the shared foundation Plans 02/03/04 all build on.
Output: Working generation pipeline, committed initial spec, two green tests, and a documented go/no-go finding on Assumption A1.
