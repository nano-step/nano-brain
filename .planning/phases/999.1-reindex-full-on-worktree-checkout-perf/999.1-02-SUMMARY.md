### Migration: `migrations/00029_add_documents_mod_time_file_size.sql`

```sql
-- +goose Up
ALTER TABLE documents
    ADD COLUMN IF NOT EXISTS mod_time  TIMESTAMPTZ,
    ADD COLUMN IF NOT EXISTS file_size BIGINT;

-- +goose Down
ALTER TABLE documents
    DROP COLUMN IF EXISTS file_size,
    DROP COLUMN IF EXISTS mod_time;
```

### Query changes: `internal/storage/queries/documents.sql`

`UpsertDocumentBySourcePath` — diff summary:
- INSERT column list: `..., supersedes_id` → `..., supersedes_id, mod_time, file_size`
- VALUES: `$9` → `$9, $10, $11`
- DO UPDATE SET: added `mod_time = EXCLUDED.mod_time`, `file_size = EXCLUDED.file_size`
- RETURNING clause unchanged

New query appended:
```sql
-- name: PreloadFileStateByWorkspace :many
SELECT source_path, mod_time, file_size, content_hash
FROM documents
WHERE workspace_hash = $1
  AND collection = $2
  AND source_path != ''
  AND mod_time IS NOT NULL
  AND file_size IS NOT NULL;
```
