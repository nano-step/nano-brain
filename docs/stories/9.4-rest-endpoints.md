## Scope (no overlap with 9.2)
1. **Config endpoints** (`internal/server/handlers/config.go`):
   - `GET /api/v1/config?include_source=bool` ŌĆö return resolved config with secrets `<redacted>`
   - `POST /api/v1/config` ŌĆö partial patch on allowlist; YAML writeback with comment preservation; trigger reload
   - `internal/config/secrets.go` ŌĆö list of secret field paths
   - `internal/config/patch.go` ŌĆö safe-patch allowlist + YAML preserving writeback (`yaml.v3` Node)
2. **Doctor endpoint** (`internal/server/handlers/doctor.go`):
   - `GET /api/v1/doctor` ŌĆö same checks as CLI in JSON; uses pure-function refactor in `internal/health/doctor/`
3. **Stats endpoint** (`internal/server/handlers/stats.go`):
   - `GET /api/v1/stats?workspace=<hash>` ŌĆö counts + breakdowns
   - Add sqlc queries: `CountDocumentsByCollection`, `CountChunksByEmbedStatus`, `CountGraphEdgesByType`, `ListTopTags`, `ListRecentDocuments`, `ListRecentQueries`
4. **Graph neighborhood dual-mode** (`internal/server/handlers/graph_neighborhood.go`):
   - `POST /api/v1/graph/neighborhood` with `node_kind=symbol|doc` discriminator
   - 500-node hard cap + truncated/frontier_nodes
   - `node_kind=doc` enriches with title/collection/updated_at via `ListDocumentsByIDs`
   - Server filters `edge_types` to `['references']` when `node_kind=doc` (defense in depth)
5. **Links endpoints** (`internal/server/handlers/links.go`):
   - `GET /api/v1/links/:doc_id/backlinks?workspace=<h>&limit=<n>` ŌĆö paginated backlinks with 200-char snippets
   - `GET /api/v1/links/resolve?workspace=<h>&query=<title-or-id>` ŌĆö wraps `internal/links.Resolver`
6. **CSRF middleware** (`internal/server/middleware/csrf.go`) ŌĆö 7-step decision order from spec; mount on POST/PUT/DELETE under `/api/v1/*`.
7. **Security headers middleware** (`internal/server/middleware/security.go`) ŌĆö `Content-Security-Policy: default-src 'self'`, `X-Content-Type-Options: nosniff`, `X-Frame-Options: DENY`, `Referrer-Policy: same-origin`. Scoped to `/ui` route group ONLY; NOT applied to `/api/v1/*`.

## Acceptance Criteria
1. All 7 endpoint groups implemented with httptest coverage.
2. CSRF 7-step matrix verified for each branch (Origin null, same-host, different-host, no-Origin-no-Referer CLI, etc.).
3. Security headers present on `/ui/*` responses; ABSENT on `/api/v1/*` responses (httptest).
4. Config patch enforces secret-field allowlist (POST with `api_key` ŌåÆ 400).
5. Doctor endpoint JSON matches CLI form.
6. Stats endpoint runs <500ms on workspaces with Ōēż1M chunks (existing indexes).
7. Graph neighborhood: symbol mode unchanged; doc mode returns docs.
8. Backlinks endpoint workspace-scoped (cross-workspace never leaks).
9. Resolve endpoint: ID vs title path; ambiguous detection.
10. `go build` + `go test -race -short ./...` pass.
11. No regression on existing endpoints.
