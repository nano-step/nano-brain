# US-9.5b React scaffold + Dashboard panel + design tokens

## Scope
1. `web/` directory at repo root — Vite 5 + React 18 + TypeScript 5 SPA
2. npm deps per design.md (NO Shiki)
3. App shell: Router (TanStack Router) + QueryClient (TanStack Query) + Theme provider
4. Design tokens (CSS variables) matching the mockup (#0D0D0D bg, #0070F3 accent, JetBrains Mono code, sharp 0-4px radii)
5. Layout: sidebar (220px) + main area + Cmd+K mount point (stub for 9.8)
6. Workspace selector (top-left of sidebar) — persists to localStorage, updates URL `?workspace=<hash>`
7. **Dashboard panel** (the only panel this story builds):
   - Calls `GET /api/v1/stats` (from 9.4)
   - Renders 6 stat cards + collections + top tags + recent docs
   - Subscribes to `/api/v1/events` (from 9.2) for live `embed_queue` counter
8. SSE hook `useEvents.ts` — single EventSource, auto-reconnect, multiplex into Zustand-equivalent store
9. Theme toggle (dark default, light optional) — persists to localStorage
10. Vite config: build output → `internal/server/webui/dist/` (consumed by 9.5a's embed.FS)
11. Makefile already done in 9.5a (web-build / web-dev / web-install / web-check)

## ACs
1. `cd web && npm ci && npm run build` succeeds
2. Total initial bundle ≤ 600 KB gzipped (verify via `vite-bundle-visualizer` or hand-check `dist/assets/*.js.gz` sizes)
3. Dashboard renders status + 6 stat cards + recent docs list when /api/v1/stats returns data
4. Workspace selector switches workspaces (URL + localStorage)
5. SSE hook reconnects on disconnect
6. Dark/light toggle works
7. First contentful paint < 500 ms on localhost (warm cache)
8. Accessibility: keyboard nav for Dashboard, focus rings visible, WCAG AA contrast on text
9. No external CDN — full UI loads with network disabled after first asset fetch
10. End-to-end: `make web-build && go build && ./nano-brain serve` → `curl http://localhost:3100/ui` → returns React app HTML (not the "Web UI not built" fallback)
11. No regression on existing Go tests

- Settings + Cmd+K + Bind banner + Destructive confirms → Story 9.8
