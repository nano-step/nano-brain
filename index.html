Use Cases

From solo developers to entire teams.

💻 Multi-machine developer

Work on office PC, home laptop, personal machine — each with different sessions. Deploy nano-brain on a VPS. Every session gets harvested. Switch machines, pick up where you left off.

Office PC ──┐ ├──► nano-brain on VPS ──► shared PostgreSQL Home Mac ───┘

👥 Team knowledge base

One server, whole team. Every developer's AI agent connects to the same PostgreSQL. Decisions, architecture notes, code intelligence — instantly shared. New hires get full context from day one.

Dev A (office) ──┐ Dev B (remote) ──┼──► nano-brain on team server Dev C (new hire) ──┘

🔍 Legacy codebase archaeology

Inherit a 5-year-old codebase with no docs? Index it. Your AI agent can now answer "what does this function do?", "why does this class exist?", "if I change this file, what else breaks?"

Go, TypeScript, Python, JavaScript, Ruby supported today. Rust, Java planned.

⚡ Pre-commit impact check

Before pushing, run memory_impact on changed files. Discover what else depends on them. Catch breaking changes before CI hits.

memory_impact("src/auth/login.ts") → Returns: auth.ts, auth.test.ts, api.ts

Query Your Codebase

Trace Call Chains

curl -X POST http://localhost:3100/api/v1/graph/trace \ -H "Content-Type: application/json" \ -d '{"workspace": "abc123", "node": "main.go::main", "max_depth": 5}'

Analyze Impact

curl -X POST http://localhost:3100/api/v1/graph/impact \ -H "Content-Type: application/json" \ -d '{"workspace": "abc123", "node": "src/auth/login.ts", "max_depth": 2}'

Generate Flow Diagrams

Get started in 3 steps

From install to your first query in under 5 minutes.

1

Install

npm install -g @nano-step/nano-brain
2

Start

nano-brain serve -d
3

Query

curl -X POST http://localhost:3100/api/v1/query \ -d '{"workspace":"abc","query":"auth flow"}'

Configuration

Config file: ~/.nano-brain/config.yml

server: host: localhost port: 3100 database: url: postgres://nanobrain:nanobrain@localhost:5432/nanobrain_dev embedding: provider: ollama url: http://localhost:11434 model: nomic-embed-text search: rrf_k: 60 recency_weight: 0.3 limit: 20

Development Setup

Run integration tests (requires PostgreSQL):

go test -race -tags=integration ./...
