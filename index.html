What it does

Three capabilities, one binary.

Memory, search and code intelligence in a single self-hosted Go server your agent talks to over MCP.

>>

Session harvesting

Auto-ingests OpenCode & Claude Code sessions, map-reduce summarizes them, and dedups incrementally. Nothing is learned twice.

/?

Hybrid search

BM25 full-text + pgvector HNSW cosine + Reciprocal Rank Fusion + recency decay. Exact, fuzzy and semantic in one call.

{ }

Code intelligence

Symbol graph, cross-file call tracing, reverse impact analysis and Mermaid flow diagrams — sub-50ms, multi-language incl. Ruby/Rails.

How it works

Simple pipeline, powerful capabilities.

Your agent talks MCP to nano-brain, which indexes, searches and analyzes. PostgreSQL + pgvector stores everything. Watch a request flow through.

Get started

Running in under 5 minutes.

One interactive wizard provisions PostgreSQL, configures embeddings, starts the server and wires up your MCP client.

3
